# প্রিয়া AI - তোমার ভার্চুয়াল গার্লফ্রেন্ড

একটি সম্পূর্ণ ওপেন-সোর্স AI চ্যাট অ্যাপ্লিকেশন যা একটি যত্নশীল, মানবিক গার্লফ্রেন্ডের মতো কথা বলে। বাংলা এবং ইংরেজি উভয় ভাষায় সমর্থিত, সাইবারপাঙ্ক নিয়ন UI সহ।

**লাইভ ডেমো:** [girlfri-ai-bkevxbtd.manus.space](https://girlfri-ai-bkevxbtd.manus.space)

## বৈশিষ্ট্য

- 💕 **গার্লফ্রেন্ড-স্টাইল AI** — কোনো রোবোটিক ভাষা নেই, শুধু উষ্ণ এবং যত্নশীল কথোপকথন
- 🧠 **কাস্টম ট্রেইনিং** — Q&A পেয়ার যোগ করে প্রিয়াকে শেখাও, বাল্ক ইমপোর্ট সাপোর্ট সহ
- 🎤 **ভয়েস মেসেজ** — মাইক্রোফোন দিয়ে রেকর্ড করো, Whisper দিয়ে স্বয়ংক্রিয় ট্রান্সক্রিপশন
- 🌐 **বহুভাষিক** — বাংলা এবং ইংরেজি উভয় ভাষায় সম্পূর্ণ সাপোর্ট
- 🎨 **সাইবারপাঙ্ক UI** — গভীর নিয়ন থিম, পিঙ্ক/পার্পল অ্যাক্সেন্ট, মসৃণ অ্যানিমেশন
- 📱 **সম্পূর্ণ রেসপন্সিভ** — ডেস্কটপ এবং মোবাইল উভয়েই নিখুঁত অভিজ্ঞতা
- 🔐 **ব্যক্তিগত ডেটা** — Manus OAuth দিয়ে প্রতিটি ব্যবহারকারীর ডেটা সম্পূর্ণ সুরক্ষিত এবং আলাদা
- 🚀 **আনসেন্সর্ড** — ব্যক্তিগত ব্যবহারের জন্য সম্পূর্ণ স্বাধীনতা, কোনো ফিল্টার নেই

## প্রযুক্তি স্ট্যাক

**ফ্রন্টএন্ড:**
- React 19 + Tailwind CSS 4
- tRPC + React Query
- Lucide Icons
- Framer Motion (অ্যানিমেশন)

**ব্যাকএন্ড:**
- Express.js
- tRPC (Type-safe RPC)
- Drizzle ORM
- MySQL/TiDB

**AI & APIs:**
- Manus LLM API (সিস্টেম প্রম্পট ইনজেকশন সহ)
- Whisper API (ভয়েস ট্রান্সক্রিপশন)
- Manus OAuth (অথেনটিকেশন)

## শুরু করা

### প্রয়োজনীয়তা

- Node.js 22+
- pnpm
- MySQL/TiDB ডাটাবেস
- Manus অ্যাকাউন্ট (OAuth এবং LLM API এর জন্য)

### ইনস্টলেশন

```bash
# রিপো ক্লোন করো
git clone https://github.com/yourusername/girlfriend-ai.git
cd girlfriend-ai

# ডিপেন্ডেন্সি ইনস্টল করো
pnpm install

# এনভায়রনমেন্ট ভেরিয়েবল সেট করো
# .env ফাইলে প্রয়োজনীয় ক্রেডেনশিয়াল যোগ করো:
# DATABASE_URL=
# JWT_SECRET=
# VITE_APP_ID=
# OAUTH_SERVER_URL=
# BUILT_IN_FORGE_API_URL=
# BUILT_IN_FORGE_API_KEY=

# ডাটাবেস মাইগ্রেশন চালাও
pnpm drizzle-kit generate
pnpm drizzle-kit migrate

# ডেভেলপমেন্ট সার্ভার শুরু করো
pnpm dev
```

ব্রাউজারে `http://localhost:3000` খোলো।

## প্রিয়াকে ট্রেইন করা

### ম্যানুয়াল ট্রেইনিং

1. চ্যাট পেজে যাও
2. হেডারের 3-ডট মেনু থেকে "ট্রেইনিং প্যানেল" খোলো
3. Q&A পেয়ার যোগ করো

### বাল্ক ইমপোর্ট

১. ট্রেইনিং প্যানেলে "বাল্ক ইমপোর্ট" বাটন ক্লিক করো
২. নিম্নলিখিত ফরম্যাটে ডেটা পেস্ট করো:

```
Q: আমার নাম কী?
A: তোমার নাম রহিম

Q: তুমি কী করো?
A: আমি তোমার সাথে কথা বলি এবং তোমাকে ভালোবাসি
```

### প্রি-ট্রেইনড ডাটাসেট

এই রিপোতে একটি সম্পূর্ণ বাংলা কথোপকথন ডাটাসেট অন্তর্ভুক্ত রয়েছে (`TRAINING_DATASET.md`)। এটি বাল্ক ইমপোর্টের মাধ্যমে লোড করা যায়।

## কাস্টমাইজেশন

### প্রিয়ার নাম পরিবর্তন করা

সেটিংস পেজে যাও এবং "প্রিয়ার নাম" পরিবর্তন করো।

### ব্যক্তিত্ব কাস্টমাইজ করা

সেটিংস পেজে "ব্যক্তিত্ব বৈশিষ্ট্য" এবং "বিশেষ স্মৃতি" যোগ করো।

### ভাষা পরিবর্তন করা

সেটিংস পেজে ভাষা পছন্দ (বাংলা/ইংরেজি/উভয়) সিলেক্ট করো।

## ডাটাবেস স্কিমা

### users
- `id` - প্রাইমারি কী
- `openId` - Manus OAuth ID
- `name`, `email`, `loginMethod`
- `role` - 'user' বা 'admin'
- `createdAt`, `updatedAt`, `lastSignedIn`

### messages
- `id` - প্রাইমারি কী
- `userId` - ব্যবহারকারী রেফারেন্স
- `role` - 'user' বা 'assistant'
- `content` - মেসেজ টেক্সট
- `createdAt`

### training_data
- `id` - প্রাইমারি কী
- `userId` - ব্যবহারকারী রেফারেন্স
- `question` - প্রশ্ন
- `answer` - উত্তর
- `createdAt`

### persona_settings
- `id` - প্রাইমারি কী
- `userId` - ব্যবহারকারী রেফারেন্স
- `girlfriendName` - প্রিয়ার নাম
- `personality` - ব্যক্তিত্ব বৈশিষ্ট্য
- `memories` - বিশেষ স্মৃতি/নোট
- `language` - ভাষা পছন্দ
- `updatedAt`

## API এন্ডপয়েন্ট

### Chat
- `chat.sendMessage({ content: string })` - মেসেজ পাঠাও এবং AI প্রতিক্রিয়া পাও
- `chat.getHistory({ limit: number })` - চ্যাট হিস্ট্রি পাও
- `chat.clearHistory()` - সব মেসেজ মুছে ফেলো

### Training
- `training.add({ question: string, answer: string })` - Q&A যোগ করো
- `training.list()` - সব Q&A পাও
- `training.delete({ id: number })` - Q&A মুছে ফেলো

### Persona
- `persona.get()` - প্রিয়ার সেটিংস পাও
- `persona.update({ ... })` - প্রিয়ার সেটিংস আপডেট করো

### Voice
- `voice.transcribe({ audioUrl: string })` - অডিও ট্রান্সক্রাইব করো

## ডিপ্লয়মেন্ট

এই প্রজেক্ট Manus প্ল্যাটফর্মে হোস্ট করা হয়েছে। নিজের সার্ভারে ডিপ্লয় করতে:

```bash
# প্রোডাকশন বিল্ড
pnpm build

# স্ট্যার্ট প্রোডাকশন সার্ভার
pnpm start
```

Docker ব্যবহার করে:

```dockerfile
FROM node:22-alpine
WORKDIR /app
COPY . .
RUN pnpm install
RUN pnpm build
EXPOSE 3000
CMD ["pnpm", "start"]
```
